## Featured Projects

### AI-Powered Research Assistant
**Technologies**: Python, TensorFlow, BERT, Flask, React  
**Duration**: 2023 - Present  
**Status**: Active Development

This project aims to revolutionize how researchers discover and consume scientific literature. The system uses advanced NLP techniques to:

- **Intelligent Paper Discovery**: Recommend relevant papers based on research interests
- **Automated Summarization**: Generate concise summaries of complex research papers
- **Research Gap Identification**: Highlight areas where additional research is needed
- **Citation Network Analysis**: Visualize connections between related works

**Key Features:**
- Natural language querying interface
- Real-time paper recommendations
- Collaborative annotation system
- Integration with major academic databases

[View on GitHub](https://github.com/bryceywang/research-assistant) | [Live Demo](https://research-assistant-demo.com)

### Smart Study Planner
**Technologies**: Python, scikit-learn, Django, PostgreSQL  
**Duration**: 2022 - 2023  
**Status**: Completed

A personalized learning system that adapts to individual study patterns and creates optimized schedules for maximum learning efficiency.

**Features:**
- Learning pattern analysis
- Adaptive scheduling algorithms
- Progress tracking and analytics
- Spaced repetition optimization
- Multi-platform synchronization

**Impact:**
- Used by 500+ students at Stanford
- Average 25% improvement in study efficiency
- Featured in Stanford Daily

[View on GitHub](https://github.com/bryceywang/study-planner) | [Case Study](https://medium.com/@bryceywang/smart-study-planner)

### Neural Network Visualizer
**Technologies**: JavaScript, D3.js, Python, PyTorch  
**Duration**: 2022  
**Status**: Completed

An interactive educational tool that helps students and researchers understand neural network architectures and their decision-making processes.

**Features:**
- Interactive network architecture visualization
- Layer-by-layer activation visualization
- Real-time weight adjustment
- Educational tutorials and explanations
- Export capabilities for presentations

**Recognition:**
- Winner of Stanford CS Education Innovation Award
- 1000+ GitHub stars
- Used in CS229 Machine Learning course

[View on GitHub](https://github.com/bryceywang/nn-visualizer) | [Live Demo](https://nn-viz.stanford.edu)
